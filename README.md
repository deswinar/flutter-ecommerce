# Flutter E-Commerce App *(WIP)*

A modern e-commerce mobile application built using **Flutter**. This project demonstrates clean architecture, responsive UI, state management, and component reusability for a seamless customer experience.

## ✨ Features

### 🛍 Customer Experience *(WIP)*
- Browse products with image, price, and ratings
- View product details with reviews, offers, and descriptions
- Add products to cart and proceed to checkout
- See personalized product suggestions

### 🧑‍💼 Owner/Admin Experience *(Upcoming)*
- Manage product listings
- Track sales and customer reviews
- Monitor stock levels and promotional offers

### 📲 UI/UX
- Bottom navigation menu with modular routing
- Responsive layouts and modern design components

## 📁 Project Structure
```
lib/
├── features/
│   ├── home/               # Home screen logic and UI
│   ├── product/            # Product list, detail, offers
│   ├── review/             # Customer reviews and ratings
│   └── ...                 # Other feature folders
├── shared/
│   ├── widgets/            # Common UI components (atoms, molecules, organisms)
│   └── utils/              # Shared helpers/utilities
├── main.dart               # App entry point
```

## 🚀 Getting Started

### Prerequisites
- Flutter SDK (latest stable)
- Dart SDK
- Android Studio / Xcode

### Installation
```bash
git clone https://github.com/deswinar/flutter-ecommerce.git
cd flutter-ecommerce
flutter pub get
flutter run
```
